⚡ PhotonFreak

🚀 Project Brief

PhotonFreak is a smart and lightweight AI-based web system for real-time prediction of solar panel output and battery state-of-health (SOH). It helps users efficiently monitor and forecast the performance of solar energy systems using minimal inputs like irradiance, temperature, voltage, and cycles.

🔍 Core Features

- Predicts solar output (in kWh) from irradiance and temperature values.
- Estimates battery SOH (%) from live sensor inputs like voltage, current, charge cycles, and temperature.
- Lightweight API-backed design, compatible with real-time embedded sensors and IoT platforms.

🎯 Use Case

Perfect for startups, solar farms, or EV battery monitoring setups needing quick and intelligent forecasts without heavy computation.

🧠 Tech Stack

- Frontend: React JS, Styled Components
- Backend: FastAPI, Scikit-learn, TensorFlow (for lightweight ML models)
- Deployment Ready: Compatible with Raspberry Pi, edge devices, or cloud APIs
